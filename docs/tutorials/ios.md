# iPhone / iPad / Mac 使用教程

::: tip 开始前
以下步骤请在[购买订阅](https://new.xiahuayun.com)后操作。
:::

## 1. 下载 Shadowrocket 客户端

Shadowrocket（小火箭）是 iOS / iPadOS 上常用的代理客户端，支持多种协议。该应用通常需要在外区 App Store 购买，售价约为 2.99 美元，需要外区 Apple ID 和可用的付款方式。

### 1.1 准备外区 Apple ID

请先登录已经购买 Shadowrocket 的外区 Apple ID。

#### iOS 26 及以上系统

打开“设置”，进入“账号”，然后在“购买与订阅”中登录外区 Apple ID。

#### iOS 26 以下系统

打开 App Store，点击右上角头像，将页面下滑到底部并退出当前账号，然后登录已经购买 Shadowrocket 的外区 Apple ID。

### 1.2 没有外区 Apple ID

如果没有可用账号，也可以参考下方共享账号页面下载：

- [Shadowrocket 共享账号](https://free.iosapp.icu)

更推荐自行注册美区独享 Apple ID，并通过礼品卡充值后购买。这样后续更新 App 会更稳定，也能减少账号失效带来的影响。

::: warning 注意事项
- 共享账号可能会频繁更改密码，如果无法下载，请更换账号或等待 30 分钟后再试。
- 使用共享账号下载的 App，后续可能无法自动更新。
- 下载完成后，请及时退出共享 Apple ID。
:::

::: danger 重要警告
- 请勿在手机“设置”中登录共享 Apple ID，仅在 App Store 中登录即可。
- 如遇安全验证，请选择“其他选项”，然后选择“不升级”。
:::

登录成功后，前往 App Store 的 Shadowrocket 商店页面，下载并安装 Shadowrocket。

![Shadowrocket 客户端下载示意](https://imgbed.foohere.de/file/1779352537162_1083badf4a38fa8ef41c4a494c9dda05.jpg)

## 2. 导入订阅配置

打开订阅仪表盘，选择一键导入，然后选择导入到 Shadowrocket。

## 3. 启动连接

回到 Shadowrocket，单击选择一个节点，然后点击右上角开关按钮启动连接。

首次启动时，系统会提示添加 VPN 配置文件，请选择同意（Allow）。

建议选择“配置模式”使用。
