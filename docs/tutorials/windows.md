# Windows 使用教程

::: tip 开始前
以下步骤请在[购买订阅](https://new.xiahuayun.com)后操作。
:::

## 1. 下载 v2rayN 电脑版

下载 v2rayN Windows 版压缩包。

- [v2rayN 下载链接](https://cloud.foohere.de/d/cloud/%E4%BB%A3%E7%90%86%E5%B7%A5%E5%85%B7/v2rayN-windows-64.zip?sign=oPdTjqM4SdhhTdBUB-30QpqI0r1H03e4kl7nTrlLZjM=:0)
- [备用下载链接](https://github.com/2dust/v2rayN/releases/download/7.20.2/v2rayN-windows-64-desktop.zip)

下载完成后，解压压缩包，然后打开 v2rayN 程序。

![下载 v2rayN](https://imgbed.foohere.de/file/1775402170970_v2-tech-01.png)

![解压 v2rayN](https://imgbed.foohere.de/file/1775402168132_v2-tech-02.png)

![打开 v2rayN 文件夹](https://imgbed.foohere.de/file/1775402172350_v2-tech-03.png)

![启动 v2rayN](https://imgbed.foohere.de/file/1775402172882_v2-tech-04.png)

## 2. 导入订阅配置

打开订阅仪表盘，复制订阅链接，然后回到 v2rayN 中添加订阅配置。

![复制订阅链接](https://imgbed.foohere.de/file/1775402170837_v2-tech-05.png)

![添加订阅](https://imgbed.foohere.de/file/1775402167132_v2-tech-06.png)

![填写订阅信息](https://imgbed.foohere.de/file/1775402170502_v2-tech-07.png)

![更新订阅](https://imgbed.foohere.de/file/1775402176006_v2-tech-08.png)

## 3. 调整代理模式

订阅导入完成后，将模式调整为“自动”。

![调整模式为自动](https://imgbed.foohere.de/file/1775402173180_v2-tech-09.png)
