# 安卓 FlClash 使用教程

::: tip 开始前
以下步骤请在[购买订阅](https://new.xiahuayun.com)后操作。
:::

## 1. 下载 FlClash 客户端

下载并安装 FlClash 客户端。

- [点击下载](https://cloud.foohere.de/d/cloud/%E4%BB%A3%E7%90%86%E5%B7%A5%E5%85%B7/FlClash-0.8.92-android-arm64-v8a.apk?sign=Ql7HqWUSgws7zQTGczYBOh1qTtMajBRP81KEdrLX3wk=:0)
- [备用下载](https://github.com/chen08209/FlClash/releases/download/v0.8.92/FlClash-0.8.92-android-arm64-v8a.apk)

下载完成后，打开安装包并完成安装。

## 2. 导入订阅配置

打开订阅仪表盘，选择一键导入到 Clash。

::: warning 导入提示
导入时不要使用任何节点，推荐使用移动数据（4G / 5G）。
:::

如果一键导入失败，请使用手动导入方式。

## 3. 手动导入订阅

打开 FlClash，点击“配置”，点击“+”，选择“URL（从 URL 导入）”。

将名称填写为“虾滑云”，然后将托管配置链接粘贴到 URL 输入框内，点击右上角“√”即可。

## 4. 启动连接

导入成功后，选择一个节点，然后启动连接。

首次启动时，系统可能会提示创建 VPN 连接，请选择允许。
