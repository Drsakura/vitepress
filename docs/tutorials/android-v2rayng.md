# 安卓 V2RayNG 使用教程

::: tip 开始前
以下步骤请在[购买订阅](https://new.xiahuayun.com)后操作。
:::

## 1. 下载 V2RayNG 客户端

下载并安装 V2RayNG 客户端。

- [点击下载](https://cloud.foohere.de/d/cloud/%E4%BB%A3%E7%90%86%E5%B7%A5%E5%85%B7/v2rayNG_2.0.18_universal.apk?sign=0mWrt1XYQBgR60kxVBwbB0BZ5lN2BGC0ovtYelwqcRs=:0)
- [备用下载](https://github.com/2dust/v2rayNG/releases/download/2.0.18/v2rayNG_2.0.18_universal.apk)

下载完成后，打开安装包并完成安装。

## 2. 复制订阅链接

打开首页，复制订阅链接。

## 3. 从剪贴板导入

打开 V2RayNG，点击右上角“+”，选择“从剪贴板导入”。

## 4. 启动连接

导入成功后，点击右下角开始按钮启动连接。
