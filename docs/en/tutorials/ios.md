# iPhone / iPad / Mac Setup Guide

::: tip Before You Start
Please complete these steps after [purchasing a subscription](https://new.xiahuayun.com).
:::

## 1. Download Shadowrocket

Shadowrocket is a commonly used proxy client for iOS and iPadOS. It supports multiple protocols. The app usually needs to be purchased from a non-mainland App Store region. The price is about USD 2.99, and you need an Apple ID for that region with an available payment method.

### 1.1 Prepare an Apple ID for Another Region

Please log in with an Apple ID that has already purchased Shadowrocket.

#### iOS 26 and Later

Open Settings, go to Account, then sign in with the Apple ID under Purchases and Subscriptions.

#### Earlier Than iOS 26

Open the App Store, tap the profile icon in the upper-right corner, scroll to the bottom, sign out of the current account, then sign in with the Apple ID that has already purchased Shadowrocket.

### 1.2 If You Do Not Have an Apple ID for Another Region

You can use the shared account page below to download the app:

- [Shadowrocket Shared Account](https://free.iosapp.icu)

We recommend registering your own US Apple ID and topping it up with a gift card before purchasing Shadowrocket. This makes future app updates more stable and reduces account availability issues.

::: warning Notes
- Shared account passwords may change often. If you cannot download the app, try another account or wait 30 minutes and try again.
- Apps downloaded with a shared account may not update automatically later.
- After downloading the app, sign out of the shared Apple ID as soon as possible.
:::

::: danger Important Warning
- Do not sign in to a shared Apple ID in the phone Settings app. Only sign in inside the App Store.
- If a security verification prompt appears, choose "Other Options", then choose "Do Not Upgrade".
:::

After signing in successfully, open the Shadowrocket App Store page, then download and install Shadowrocket.

## 2. Import the Subscription

Open your subscription dashboard, choose one-click import, then choose to import into Shadowrocket.

## 3. Start the Connection

Return to Shadowrocket, tap a node to select it, then tap the switch button in the upper-right corner to start the connection.

When starting for the first time, the system will ask to add a VPN configuration. Choose Allow.

We recommend using Configuration mode.
