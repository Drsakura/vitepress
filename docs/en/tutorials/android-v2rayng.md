# Android V2RayNG Setup Guide

::: tip Before You Start
Please complete these steps after [purchasing a subscription](https://new.xiahuayun.com).
:::

## 1. Download V2RayNG

Download and install the V2RayNG client.

- [Download](https://cloud.foohere.de/d/cloud/%E4%BB%A3%E7%90%86%E5%B7%A5%E5%85%B7/v2rayNG_2.0.18_universal.apk?sign=0mWrt1XYQBgR60kxVBwbB0BZ5lN2BGC0ovtYelwqcRs=:0)
- [Backup Download](https://github.com/2dust/v2rayNG/releases/download/2.0.18/v2rayNG_2.0.18_universal.apk)

After the download is complete, open the APK file and finish the installation.

## 2. Copy the Subscription Link

Open the homepage and copy the subscription link.

## 3. Import from Clipboard

Open V2RayNG, tap "+" in the upper-right corner, then choose "Import from Clipboard".

## 4. Start the Connection

After the import succeeds, tap the start button in the lower-right corner.
