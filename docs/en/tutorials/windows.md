# Windows Setup Guide

::: tip Before You Start
Please complete these steps after [purchasing a subscription](https://new.xiahuayun.com).
:::

## 1. Download v2rayN for Windows

Download the v2rayN Windows zip package.

- [v2rayN Download](https://cloud.foohere.de/d/cloud/%E4%BB%A3%E7%90%86%E5%B7%A5%E5%85%B7/v2rayN-windows-64.zip?sign=oPdTjqM4SdhhTdBUB-30QpqI0r1H03e4kl7nTrlLZjM=:0)
- [Backup Download](https://github.com/2dust/v2rayN/releases/download/7.20.2/v2rayN-windows-64-desktop.zip)

After the download is complete, extract the zip package, then open v2rayN.

![Download v2rayN](https://imgbed.foohere.de/file/1775402170970_v2-tech-01.png)

![Extract v2rayN](https://imgbed.foohere.de/file/1775402168132_v2-tech-02.png)

![Open the v2rayN folder](https://imgbed.foohere.de/file/1775402172350_v2-tech-03.png)

![Start v2rayN](https://imgbed.foohere.de/file/1775402172882_v2-tech-04.png)

## 2. Import the Subscription

Open your subscription dashboard, copy the subscription link, then add the subscription in v2rayN.

![Add subscription](https://imgbed.foohere.de/file/1775402167132_v2-tech-06.png)

## 3. Set Proxy Mode

After importing the subscription, set the mode to "Auto".
