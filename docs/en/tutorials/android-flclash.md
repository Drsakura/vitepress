# Android FlClash Setup Guide

::: tip Before You Start
Please complete these steps after [purchasing a subscription](https://new.xiahuayun.com).
:::

## 1. Download FlClash

Download and install the FlClash client.

- [Download](https://cloud.foohere.de/d/cloud/%E4%BB%A3%E7%90%86%E5%B7%A5%E5%85%B7/FlClash-0.8.92-android-arm64-v8a.apk?sign=Ql7HqWUSgws7zQTGczYBOh1qTtMajBRP81KEdrLX3wk=:0)
- [Backup Download](https://github.com/chen08209/FlClash/releases/download/v0.8.92/FlClash-0.8.92-android-arm64-v8a.apk)

After the download is complete, open the APK file and finish the installation.

## 2. Import the Subscription

Open your subscription dashboard and choose one-click import to Clash.

::: warning Import Tip
Do not use any node while importing. We recommend using mobile data, such as 4G or 5G.
:::

If one-click import fails, use the manual import method below.

## 3. Manually Import the Subscription

Open FlClash, tap "Config", tap "+", then choose "URL".

Set the name to "Xiahuayun", paste the hosted configuration link into the URL field, then tap the checkmark in the upper-right corner.

## 4. Start the Connection

After the subscription is imported, select a node and start the connection.

When starting for the first time, the system may ask to create a VPN connection. Choose Allow.
